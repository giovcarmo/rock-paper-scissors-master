# Rock, Paper, Scissors (Frontend Mentor)

Para praticar o módulo de TypeScript, fiz um exercício do frontend mentor que pede para criar um joguinho de pedra papel tesoura, é simples o design, porem um pouco complexo, ele trás um botão que abre uma janela com as regras, e a interação entre os botões que recebe padrões diferente de tamanho após a escolha do jogador. Tem também a parte da dinâmica do jogo, onde cada cenário é um estágio diferente que pede uma configuração diferente de renderização. Foi ótimo desenvolver este simples mas complexo desafio, e praticar TypeScript.

## Sobre
Não tive tantas dificuldades mas fiz tive de usar a IA algumas vezes para tirar dúvidas e resolver alguns problemas, com isso aprendi bastante, pois tive de pensar nos cenários mas não estava dando certo, até que então fui fragmentando os problemas até resolver o máximo possível sem ajuda. Eu usei ao invés do JS o TS que acabei de estudar no curso, para reforçar e praticar.
